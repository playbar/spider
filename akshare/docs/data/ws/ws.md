## [AkShare](https://github.com/jindaxiang/akshare) 板块实时监控

### 商品、外汇、股市、美股、国债、指数

接口: watch

目标地址: https://datacenter.jin10.com/market

描述: 获取商品、外汇、股市、美股、国债、指数实时行情数据, 如需要存储数据请修改 **on_message** 接口

限量: 主动推送

输入参数

| 名称   | 类型 | 必选 | 描述                                                                              |
| -------- | ---- | ---- | --- |
| - | -  | -    |  -|

输出参数

| 名称          | 类型 | 默认显示 | 描述           |
| --------------- | ----- | -------- | ---------------- |
| -     | -   | -        | 基于 websocket 的接口  |  

接口示例

```python
import akshare as ak
ak.watch()
```
